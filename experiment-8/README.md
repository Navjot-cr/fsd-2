## Aim
To develop RESTful APIs using a backend framework.

## Technology Used
- Node.js
- Express.js
- Body-parser

## Installation

1. Clone or download the project
2. Install dependencies:

npm install

3. Run the server:

node app.js

## API Endpoints

### Get all students
GET /students

### Get student by ID
GET /students/:id

### Add student
POST /students

### Update student
PUT /students/:id

### Delete student
DELETE /students/:id

## Output
The API performs CRUD operations and returns data in JSON format.
